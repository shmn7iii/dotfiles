# dotfiles

### mac OS

```bash
$ xcode-select --install
$ git clone https://github.com/shmn7iii/dotfiles.git ~/.dotfiles && cd .dotfiles
$ /bin/bash setup.macos.sh
```

```bash
# option: mouse speed
$ defaults write "Apple Global Domain" com.apple.mouse.scaling 10
```

### Manjaro Linux

```bash
$ git clone https://github.com/shmn7iii/dotfiles.git ~/.dotfiles && cd .dotfiles
$ /bin/bash setup.manjaro.sh
```
